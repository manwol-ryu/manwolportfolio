site.json 시각 편집기

이 페이지는 data/site.json을 읽고, JSON을 미리 보고 수정한 뒤 파일로 내려받을 수 있습니다. GitHub Pages 같은 정적 호스팅 환경에서는 자동 저장/커밋이 불가능하므로 변경 사항은 로컬에서 직접 저장 후 푸시해야 합니다.

보안 주의

공개 저장소에 배포된 정적 페이지는 누구나 접근할 수 있습니다. 실제 관리자 인증이 필요하다면 서버 기반 인증/권한 관리가 필요합니다.

사용 방법

  • 먼저 JSON 불러오기를 눌러 현재 데이터를 로드합니다.
  • 필요한 내용을 수정한 뒤 JSON 다운로드로 파일을 저장하세요.
  • 변경된 파일은 로컬에서 다시 저장하거나 GitHub에서 직접 커밋해야 배포됩니다.
